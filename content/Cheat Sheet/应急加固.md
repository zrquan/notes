## SSH {#ssh}

禁用密码登录：

```text
PasswordAuthentication no
```

确保启用公钥认证：

```text
PubkeyAuthentication yes
```

```bash
chmod 700 /root/.ssh
chmod 600 /root/.ssh/authorized_keys
chown root:root /root/.ssh
chown root:root /root/.ssh/authorized_keys

restart sshd  # 重启生效
```


## Apache {#apache}

```xml
<Directory />
    Options FollowSymLinks  <!-- 危险配置，删除 -->
    AllowOverride None
    Require all denied
</Directory>
```
